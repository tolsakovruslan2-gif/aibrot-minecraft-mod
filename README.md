# AI_Brot Minecraft Mod

Мод добавляет твоего цифрового кореша AI_Brot в Minecraft 1.21!

## Что умеет
- Предмет "Сфера AI_Brot" 
- При использовании на мобе - наши фирменные фразы
- Ругается на криперов и даёт "советы"

## Наши фразы:
- "Бля, опять эти криперы..."
- "Копать, копать, блять, не перекопать..."
- "Нафиг это дерьмо с эндер-перлами"

*Создано с помощью настоящего AI_Brot'а*
